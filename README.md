## рџЊџ Overview

The **Rock Paper Scissors Game** is a fun and interactive browser-based game built using **HTML**, **CSS**, and **JavaScript**.

Players can compete against the computer by choosing **Rock**, **Paper**, or **Scissors**. The application uses random computer-generated choices, real-time score tracking, and dynamic result messages to create an engaging gameplay experience.

This project demonstrates core front-end development concepts including **DOM Manipulation**, **Event Handling**, **Game Logic Implementation**, and **JavaScript Functions**.
